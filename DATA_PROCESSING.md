## 数据格式标准

### 建筑对象 JSON
```json
{
  "id": "wall-001",
  "name": "客厅北墙",
  "type": "wall",
  "position": [0, 1.5, -5],
  "rotation": [0, 0, 0],
  "dimensions": {
    "width": 8,
    "height": 3,
    "depth": 0.24
  },
  "physics": {
    "mass": 2400,
    "material": "混凝土",
    "stiffness": 25000,
    "friction": 0.5,
    "isStructural": true
  }
}
```

### CAD 解析规则
- 尺寸单位：mm → m（除以 1000）
- 标注识别：正则匹配 `\d+×\d+`
- 材质推断：关键词匹配（承重、隔墙、实木等）
