# 具身智能模型测试平台 — 竞品/合作分析

> 来源：用户2026-04-24提供的完整产品文档
> 研发方：上海科技大学顾家远实验室 + 上海本原智数科技有限公司

---

## 1. 产品定位

**一站式具身智能评测解决方案**：仿真引擎 + 标准化测试 + 开放数据集 + 算法竞技场

核心解决三大痛点：
- 开发周期长（算法→实机 12-18个月）
- 测试成本高（100㎡测试场 >500万元）
- 数据不一致（缺乏横向对比标准）

---

## 2. 技术架构（与我们项目对比）

| 模块 | 本原智数方案 | 我们的方案 | 关联度 |
|------|------------|-----------|--------|
| 3D场景生成 | 3D Gaussian Splatting + 3D AIGC | ✅ 已有高斯泼斯（3场景训练完成） | 🔴 直接重叠 |
| 场景布局 | 深度估计+位姿估计 → 真实物体布局 | ✅ 151门/窗对象+CAD几何约束 | 🔴 直接重叠 |
| 语义标注 | 多模态大模型自动标注 | ✅ 场景分类体系+规则库 | 🟡 部分重叠 |
| 仿真适配 | 系统辨识对齐sim-real | ⚠️ Psi-W0仿真器（基础版） | 🟡 需加强 |
| 数据闭环 | fine-tune → 评测 → 采集 → 迭代 | ✅ Psi-W0数据飞轮 | 🔴 理念一致 |
| 评测基准 | 竞技场+排行榜+多维指标 | ❌ 无 | 差异化方向 |
| 多机器人支持 | Franka/UR5/xArm/宇树H1/傅利叶GR-1 | ⚠️ 仅越疆CR系列 | 🟡 需扩展 |

## 4. 创新亮点（3个）

1. **仿真数据集对齐训练+评测范式** — 避免直接解决sim-to-real gap
2. **多模态AIGC场景生成引擎** — 高斯泼溅+3D生成+真实布局
3. **数据闭环生产系统** — 预训练→微调→评测→采集迭代

## 6. 对我们项目的启示

### 🔴 直接竞争领域
- 3D场景生成（高斯泼溅）— 我们已有3个场景训练完成
- 数据飞轮 — 我们Psi-W0已有基础架构
- 建筑场景仿真 — 我们的差异化优势（建筑垂直领域）

### 🟢 我们的差异化优势
1. **建筑垂直领域** — 他们做通用场景，我们深耕建筑（CAD/VR/施工）
2. **真实建筑数据** — 76个CAD+1027效果图+709VR，他们需从零构建
3. **物理世界模型** — 我们有SpatialEncoder 98.1%、PhysicsMLP等
4. **具身施工任务** — 钻孔/搬运/清洁等建筑专用任务

### 🟡 需要补齐的短板
1. **评测系统** — 我们没有标准化评测基准
2. **多机器人适配** — 我们仅适配越疆
3. **sim-to-real对齐** — 缺少系统辨识模块
4. **数据扩增** — 缺少ROSIE/DIAL/MimicGen等自动扩增

### 💡 潜在合作点
1. 我们提供建筑场景数据 → 他们接入评测平台
2. 他们的评测基准 → 我们验证世界模型性能
3. 联合制定建筑领域具身智能评测标准
