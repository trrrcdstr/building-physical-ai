## 一、具身智能定义与范畴

### 1.1 核心概念
- **具身人工智能(EAI/Embodied AI)**: 让AI拥有物理身体，能够在真实物理世界中感知、理解、推理、交互和操作
- **Physical AI**: 数字AI向物理世界的延伸，是具身智能的核心方向
- **VLA (Vision-Language-Action)**: 视觉-语言-动作多模态大模型
- **数字孪生**: 物理世界的虚拟镜像

### 1.2 具身智能 vs 传统AI

| 维度 | 传统AI | 具身智能 |
|------|--------|----------|
| 感知 | 2D图像/文本 | 3D视觉+触觉+力觉 |
| 决策 | 静态推理 | 动态交互 |
| 执行 | 无 | 机器人/AGV/无人机 |
| 物理理解 | 缺失 | Sim2Real+物理引擎 |

## 二、技术架构

### 2.1 具身智能系统分层

```
感知层: RGB-D | LIDAR | 触觉 | IMU
     ↓
理解层: 3D重建 | 场景理解 | 物体识别
     ↓
推理层: LLM | VLA | 任务规划
     ↓
控制层: 运动控制 | 灵巧操作 | 力控
     ↓
执行层: 机械臂 | 移动平台 | 灵巧手
```

## 三、建筑领域应用场景

### 3.1 建筑施工机器人任务

| 任务类型 | 具体操作 | 物理要求 |
|----------|----------|----------|
| 钻孔 | 墙面钻孔安装挂件 | 力控<30N，精度±2mm |
| 搬运 | 建材/家具搬运 | 负载5-20kg |
| 清洁 | 地面/墙面清洁 | 持续力10-50N |
| 安装 | 门窗/灯具安装 | 多步骤协同 |
| 巡检 | 工地安全巡检 | 移动+感知 |

### 3.2 建筑场景特点

- **空间复杂**: 室内/室外/高低空多层次
- **物理约束多**: 承重墙/管线/防火分区
- **工具多样**: 电钻/螺丝刀/锯/焊机
- **任务长链**: 多步骤协同操作
- **安全要求高**: 人机共存环境下安全

### 3.3 建筑领域物理AI的特殊性

1. **大尺度空间**: 房间级到建筑级空间感知
2. **刚性物体操作**: 木材/金属/混凝土
3. **非结构化环境**: 施工现场vs已装修环境
4. **多机器人协同**: 多个机器人分工协作
5. **工具使用**: 需掌握多种建筑工具使用

## 四、人才与市场

### 4.1 行业规模

- 全球具身智能人才缺口: 5000+ (白皮书数据)
- 中国具身智能市场规模: 快速增长中
- 关键岗位: 机器人工程师/感知算法/运动控制

### 4.2 产业链图谱

```
上游: 核心零部件(电机/减速器/传感器)
中游: 机器人本体(机械臂/移动平台/灵巧手)
下游: 系统集成(建筑/工厂/物流/医疗)
```

## 六、建筑Physical AI技术路线

### 6.1 三层架构

```
L1 感知层: VR全景(3DGS) + CAD几何 + 传感器融合
L2 认知层: 物理常识(PhysicsMLP) + 空间关系(SpatialEncoder)
L3 规划层: VLA指令理解 + 任务规划 + 运动控制
```

### 6.2 核心创新

1. **建筑领域VLA**: 针对建筑任务的专用动作空间
2. **物理引擎集成**: 碰撞检测/力控仿真/轨迹优化
3. **知识图谱增强**: 建筑规范+机电系统+材料属性
4. **Sim2Real飞轮**: 仿真数据→真实场景迁移
