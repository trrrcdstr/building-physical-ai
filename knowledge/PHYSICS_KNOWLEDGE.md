|------|-------------|-------------|--------|------------|
| **抛光砖** | 2000-2300 | 6-7 | <0.1% | 滑，需防滑抓取 |

### 摩擦属性

| 接触面 | 静摩擦系数 μs | 动摩擦系数 μk | 机器人应用 |
|--------|--------------|--------------|------------|
| **瓷砖-橡胶吸盘** | 0.4-0.6 | 0.3-0.5 | 吸盘抓取 |
| **瓷砖-金属夹具** | 0.2-0.4 | 0.15-0.3 | 需软垫 |
| **瓷砖-瓷砖** | 0.3-0.5 | 0.2-0.4 | 堆叠稳定性 |
| **瓷砖-TPE手套** | 0.4-0.6 | 0.3-0.5 | 灵巧手抓取 |

**抓取力计算：**
```
F_grip > m × g / (2 × μs)

# 例：800×800瓷砖, 厚度12mm
# 体积 = 0.8 × 0.8 × 0.012 = 0.00768 m³
# 质量 = 2300 × 0.00768 = 17.7 kg
# μs = 0.5 (橡胶吸盘)
# F_grip > 17.7 × 9.8 / (2 × 0.5) = 173.5 N
```

## 📐 物理计算示例

### 示例1：瓷砖抓取力计算

```python
# 800×800×12mm 瓷砖
length = 0.8    # m
width = 0.8     # m
thickness = 0.012  # m
density = 2300  # kg/m³
mu_s = 0.5      # 橡胶吸盘摩擦系数

# 计算
volume = length * width * thickness  # 0.00768 m³
mass = density * volume  # 17.7 kg
weight = mass * 9.8  # 173.5 N

# 最小抓取力（单吸盘）
F_grip_min = weight / (2 * mu_s)  # 173.5 N

# 安全系数 2.0
F_grip_safe = F_grip_min * 2.0  # 347 N

print(f"瓷砖质量: {mass:.1f} kg")
print(f"安全抓取力: {F_grip_safe:.0f} N")
```

### 示例2：脆性物体搬运约束

```python
# 瓷砖脆性约束
max_span = 0.3  # 最大悬空跨度 m
max_drop = 0.05  # 最大跌落高度 m
max_impact = 50  # 最大冲击力 N

# 搬运速度限制
# 冲击力 F = m × a
# 若跌落5cm: a = √(2×g×h) / t
# 限制加速度 < 3g

max_acceleration = 3 * 9.8  # 29.4 m/s²
print(f"搬运加速度限制: {max_acceleration:.1f} m/s²")
```

## ⚠️ 机器人安全提示

### 瓷砖搬运
- ✅ 使用双吸盘+支撑
- ✅ 悬空跨度 <30cm
- ✅ 加速度 <3g
- ❌ 禁止单手抓取大尺寸瓷砖
- ❌ 禁止跌落 >5cm

### 木地板搬运
- ✅ 长板竖直搬运
- ✅ 堆叠不超过5块
- ✅ 注意企口保护
- ❌ 禁止过度弯曲

### 门板搬运
- ✅ 双人/双手机器人
- ✅ 支撑底部
- ✅ 注意铰链位置
- ❌ 禁止拖拽表面
