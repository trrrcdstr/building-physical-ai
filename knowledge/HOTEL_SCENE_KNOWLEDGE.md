## 🌍 世界模型映射

### 酒店场景 → 4D世界模型

```
酒店 → World Model Layers

L1_几何层:
  - 客房布局: room[id, polygon, floor]
  - 走廊网络: corridor[path, width, height]
  - 电梯位置: elevator[id, position, floors]
  - 楼梯位置: stair[id, position, floors]

L2_语义层:
  - 功能分区: {大堂, 客房, 餐饮, 宴会, 康体, 后勤}
  - 客房等级: {标准间, 商务房, 套房, 总统套房}
  - 服务动线: service_paths[任务, 路径]

L3_物理层:
  - 地面材质: {地毯, 瓷砖, 木地板} → 摩擦系数
  - 墙面材质: {墙纸, 木饰面, 软包}
  - 设备参数: {空调, 热水, 照明}
  - 门禁系统: access_control[区域, 权限]

L4_行为层:
  - 入住流程: check_in_flow
  - 服务流程: service_flow[任务]
  - 清洁计划: cleaning_schedule
  - 维护计划: maintenance_schedule
```
