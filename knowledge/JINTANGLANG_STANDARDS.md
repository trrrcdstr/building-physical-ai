## 核心工艺详解

### 1. 前期勘测流程

```
Step1: 墙面检测
├── 检查墙面是否有空鼓、可选修补
├── 检查墙面的垂直度、平整度、偏差情况
└── 特殊情况一定要业主/油漆公司确认

Step2: 前期底盒检测
├── 检查底盒是否采用连体排布/独立排布方式
├── 检查底盒的"水平度/平整度偏差"是否符合要求
└── 特殊情况一定要业主/油漆公司确认
```

### 2. 水电工艺规范

#### 给水路线预埋
- 水管材质：PPR管
- 水管直径：DN20/DN25
- 打压压力：0.8MPa（8kg）
- 稳压时间：30分钟

#### 强弱电工艺
- 强电：红色
- 弱电：蓝色
- 地线：黄绿双色
- 线管间距：≥20mm

### 3. 防水工艺

#### 丙纶布防水
```
材料：400g丙纶布
工艺：两布三油
步骤：
1. 清理基层
2. 涂刷水泥浆
3. 铺贴丙纶布
4. 涂刷水泥浆
5. 铺贴第二层
6. 蓄水试验
```

#### 卫生间防水标准
- 淋浴区防水高度：≥1800mm
- 其他区域防水高度：≥300mm
- 门口防水翻边：≥500mm
- 闭水试验时间：≥48小时

### 4. 砌墙工艺

#### 标准放线
- 激光水平仪定位
- 墨斗弹线
- 红外线复核

#### 灰缝标准
- 灰缝厚度：8-12mm
- 灰缝饱满度：≥80%
- 垂直度偏差：≤5mm/m

### 5. 吊顶工艺

#### 石膏板吊顶
```
材料：9.5mm石膏板
龙骨：轻钢龙骨
间距：主龙骨≤900mm
      副龙骨≤300mm
螺钉间距：150-170mm
```

#### 转角工艺（防开裂）
- L型整板切割
- 转角处禁止拼接
- 双层石膏板加固

### 6. 油工工艺

#### 墙面处理流程
```
墙固（界面剂）→ 石膏找平 → 腻子一遍 → 腻子二遍 → 打磨 → 底漆 → 面漆
```

#### 阴阳角处理
- 材料：PVC阴阳角条
- 垂直度：≤2mm/2m
- 腻子厚度：≤3mm

## 机器人可理解的工艺知识

### 场景：卫生间防水检测

```yaml
任务: 检测卫生间防水质量
前置条件:
  - 防水层已施工完成
  - 闭水试验未开始或进行中
  
步骤:
  1. 定位卫生间门口
  2. 检测地面是否有积水
  3. 测量水面高度
  4. 检查墙角防水翻边高度
  5. 记录检测数据
  
物理知识:
  - 防水高度: 淋浴区≥1800mm, 其他≥300mm
  - 闭水试验: 注水深度100-200mm
  - 检测时间: ≥48h
  
异常处理:
  - 发现渗漏: 标记位置，上报
  - 水量减少: 可能渗漏，需复查
```

### 场景：石膏板吊顶安装检测

```yaml
任务: 检测石膏板吊顶安装质量
前置条件:
  - 石膏板已安装
  - 螺钉已固定
  
检测项:
  - 螺钉间距: 标准150-170mm
  - 转角处理: L型整板，禁止拼接
  - 板缝处理: 嵌缝石膏+网格带
  
物理参数:
  - 螺钉沉入深度: 1-2mm
  - 板缝宽度: 5-8mm
  - 平整度: ≤3mm/2m
```

## 质量验收标准

### A级标准（优良）
- 观感质量：光滑、流畅、色泽一致
- 实测偏差：≤允许偏差的50%
- 资料完整：齐全、真实、有效

### B级标准（合格）
- 观感质量：基本光滑、流畅
- 实测偏差：≤允许偏差
- 资料完整：基本齐全

### C级标准（不合格）
- 观感质量：粗糙、明显缺陷
- 实测偏差：超过允许偏差
- 资料问题：缺失严重
