# 智能世界2035 - 核心洞察

> 来源：智能世界2035 报告
> 关键主题：迈向通用人工智能（AGI）

---

## 核心观点

### 1. AGI 的必由之路：走向物理世界

> 迈向通用人工智能，关键在于走向物理世界。

**当前瓶颈：**
- AI 在感知和生成上取得重大成就
- 但仍困于基于多重线性统计关联的框架
- 缺乏对复杂物理现象以及因果关系的深层推理能力

**突破方向：**
- 数字世界与物理世界深度融合的新架构
- 数理原理或知识内嵌到数据空间
- 实现"精准智能"

### 2. 三大技术支柱

| 支柱 | 说明 |
|------|------|
| **感知技术** | 新一代传感器（宏观+微观+高动态+多模态） |
| **模型算法** | 向脑智能学习，突破现有范式 |
| **算力芯片** | 光计算、量子计算、存算一体等新型范式 |

### 3. 五大基础技术融合

```
通信与网络
    ↓
感知与交互 → 存储与计算
    ↓              ↓
    └──→ 新能源 ←┘
            ↓
        智能技术
            ↓
        超级智能体
```

## Physical AI 的定位

### 数字 AI vs 物理 AI

| 数字 AI | 物理 AI |
|---------|---------|
| 处理信息世界 | 理解物理世界并交互 |
| 数据驱动 | 物理规律驱动 |
| 感知 + 生成 | 感知 + 理解 + 执行 |
| 虚拟决策 | 实体行为 |

### 物理 AI 的核心能力

```
感知 → 理解 → 决策 → 执行 → 反馈
  ↓       ↓       ↓       ↓       ↓
传感器  因果推理 世界模型 控制器 闭环学习
```

---

## 世界模型的关键作用

### 空间智能

让 AI 摆脱二维平面，进入三维立体世界：
- 场景感知（物体位置识别）
- 任务规划（路径设计）
- 自主实施（动作执行）

### 具身智能

AI 不再只存在于服务器里，而是有了"身体"：
- 实体交互
- 物理规律理解
- 通过试错学习

### 世界模型 = 心中的"模拟器"

```
预测未来：击球手的大脑预测球轨迹
         ↓
   在"脑海"中进行思维实验
         ↓
   预测不同行动的结果
```

## 建筑领域的 Physical AI

### 我们的定位

```
智能世界2035
    ↓
Physical AI（各行业）
    ↓
建筑领域 Physical AI
    ↓
世界模型 + AI Agent + 执行引擎
```

### 建筑 Physical AI 的价值

| 传统方式 | 我们的方式 |
|---------|-----------|
| 数字孪生"看"不能"推演" | 4D 世界模型 + 因果推理 |
| 机器人不懂建筑空间 | CAD 施工图注入物理常识 |
| 仿真成本高、周期长 | 工程数据直接构建模型 |
| 不可泛化 | 物理规则通用化 |

### 建筑场景应用

1. **智能家居**
   - 机器人理解空间、电路、水路
   - 自主完成任务（开灯、开门、洗碗）

2. **施工仿真**
   - 安全模拟
   - 风险预判

3. **运维管理**
   - 设备巡检
   - 预测性维护
