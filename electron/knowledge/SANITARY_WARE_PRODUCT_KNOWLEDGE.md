**马桶坑距选择指南（用于机器人安装定位）：**

```
坑距定义：排水管中心到墙壁的距离
305mm坑距 → 适配大多数新房（开发商标准）
400mm坑距 → 老房改造常见
180mm坑距 → 壁挂式马桶（需隐藏水箱）

安装检查清单：
1. 测量实际坑距（±5mm容差）
2. 确认地面防水层状态
3. 检查进水角阀位置（左侧45cm高度）
4. 预留电源插座（右侧30cm，高度50cm）
5. 确认排污管伸出地面高度（≤20mm）
```

## 三、空间解决方案（工程应用标准布局）

### 3.1 商业卫生间标准平面布局

**大型交通枢纽卫生间（以P17/P21为参考）：**

```
典型平面尺寸：26600×5901mm（约26.6m×5.9m）
分区：男卫 / 女卫 / 第三卫生间 / 母婴室

男卫布局要素：
- 小便斗区：深度≥600mm/个，间距≥600mm
- 洗手盆区：台面高度800mm，深度550mm
- 大便器区：厕格净尺寸≥900×1200mm（有门）/ ≥900×1500mm（无障碍）
- 儿童小便斗：高度400mm，距地

产品推荐：
洗手盆：AG5706A/B（大尺寸台下盆）
水龙头：ALD51009D（感应龙头，节水）
小便斗：AG7113AB
大便器：AD810A + AG4316AUCP（挂墙砖）
地漏：AF3507CP（线型）
隔断：ARM23820CP（8mm钢化玻璃）
```

**大型商场卫生间（以P21为参考）：**

```
标准厕格尺寸：宽900mm × 深1400mm × 高2100mm
门与隔板间隙：25~35mm
隔板厚度：12mm
地坪标高差：无障碍厕格入口≤20mm

容纳人数计算（参考）：
- 女性：按男女人数比1:1.2，每70人配置1个大便器
- 男性：每70人配置1个大便器+2个小便斗
- 洗手盆：每3个大便器配1个洗手盆
```

**医院卫生间（以P23-P25为参考）：**

```
设计要点：
- 轮椅回转直径：≥1500mm
- 安全抓杆：直径32mm，承重≥250kg，距地700mm
```
