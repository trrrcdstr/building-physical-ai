## 核心工艺详解

### 1. 前期勘测流程

```
Step1: 墙面检测
├── 检查墙面空鼓情况
├── 测量垂直度、平整度
└── 记录偏差数据

Step2: 底盒检测
├── 检查底盒排布方式
├── 检测水平度/平整度
└── 复核位置尺寸
```

### 2. 水电工艺规范

#### 给水路线预埋
| 参数 | 标准值 |
|------|--------|
| 水管材质 | PPR管 |
| 冷水管直径 | DN20 |
| 热水管直径 | DN25 |
| 打压压力 | 0.8MPa |
| 稳压时间 | 30分钟 |

#### 强弱电工艺
| 类型 | 颜色 | 用途 |
|------|------|------|
| 强电 | 红色 | 照明、插座 |
| 弱电 | 蓝色 | 网络、电视 |
| 地线 | 黄绿双色 | 保护接地 |

### 3. 防水工艺

#### 丙纶布防水
```
材料规格：400g/m²
施工工艺：两布三油
标准流程：
1. 基层清理
2. 涂刷水泥浆
3. 铺贴丙纶布
4. 再次涂刷
5. 第二层布
6. 蓄水试验
```

#### 防水高度标准
| 区域 | 防水高度 | 标准 |
|------|----------|------|
| 淋浴区 | ≥1800mm | 国标 |
| 洗手台 | ≥1000mm | 国标 |
| 其他区域 | ≥300mm | 国标 |
| 门口翻边 | ≥500mm | 国标 |
| 闭水试验 | ≥48h | 无渗漏 |

### 4. 砌墙工艺

#### 标准放线
- 工具：激光水平仪
- 方法：墨斗弹线
- 复核：红外线检测

#### 灰缝标准
| 参数 | 标准值 |
|------|--------|
| 灰缝厚度 | 8-12mm |
| 灰缝饱满度 | ≥80% |
| 垂直度偏差 | ≤5mm/m |

### 5. 吊顶工艺

#### 石膏板吊顶
| 参数 | 标准值 |
|------|--------|
| 石膏板厚度 | 9.5mm |
| 主龙骨间距 | ≤900mm |
| 副龙骨间距 | ≤300mm |
| 螺钉间距 | 150-170mm |

#### 转角防裂工艺
- L型整板切割
- 转角处禁止拼接
- 双层石膏板加固

### 6. 油工工艺

#### 墙面处理流程
```
墙固（界面剂）→ 石膏找平 → 腻子一遍 → 腻子二遍 → 打磨 → 底漆 → 面漆
```

#### 阴阳角处理
| 参数 | 标准值 |
|------|--------|
| 材料 | PVC角条 |
| 垂直度 | ≤2mm/2m |
| 腻子厚度 | ≤3mm |

## 机器人可理解的工艺知识

### 场景：卫生间防水检测

```yaml
任务: 检测卫生间防水质量
前置条件:
  - 防水层已施工完成
  - 闭水试验进行中
  
检测步骤:
  1. 定位卫生间门口
  2. 检测地面是否有积水
  3. 测量水面高度
  4. 检查墙角防水翻边高度
  5. 记录检测数据
  
物理知识:
  - 防水高度: 淋浴区≥1800mm, 其他≥300mm
  - 闭水试验: 注水深度100-200mm
  - 检测时间: ≥48h
  
异常处理:
  - 发现渗漏: 标记位置，上报
  - 水量减少: 可能渗漏，需复查
```

### 场景：石膏板吊顶安装检测

```yaml
任务: 检测石膏板吊顶安装质量
前置条件:
  - 石膏板已安装
  - 螺钉已固定
  
检测项:
  - 螺钉间距: 标准150-170mm
  - 转角处理: L型整板，禁止拼接
  - 板缝处理: 嵌缝石膏+网格带
  
物理参数:
  - 螺钉沉入深度: 1-2mm
  - 板缝宽度: 5-8mm
  - 平整度: ≤3mm/2m
```

### 场景：瓷砖铺贴检测

```yaml
任务: 检测瓷砖铺贴质量
检测项:
  - 瓷砖平整度: ≤2mm/2m
  - 留缝宽度: 2-3mm（根据瓷砖规格）
  - 阴阳角方正度: ≤3mm
  - 空鼓率: ≤5%
  
物理参数:
  - 水泥砂浆比例: 1:3
  - 瓷砖吸水率: <0.5%
  - 粘结强度: ≥0.5MPa
```

## 质量验收标准

### A级标准（优良）
| 项目 | 标准 |
|------|------|
| 观感质量 | 光滑、流畅、色泽一致 |
| 实测偏差 | ≤允许偏差的50% |
| 资料 | 齐全、真实、有效 |

### B级标准（合格）
| 项目 | 标准 |
|------|------|
| 观感质量 | 基本光滑、流畅 |
| 实测偏差 | ≤允许偏差 |
| 资料 | 基本齐全 |

### C级标准（不合格）
| 项目 | 标准 |
|------|------|
| 观感质量 | 粗糙、明显缺陷 |
| 实测偏差 | 超过允许偏差 |
| 资料 | 缺失严重 |
