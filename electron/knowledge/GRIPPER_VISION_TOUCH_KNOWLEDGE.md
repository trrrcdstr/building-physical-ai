## 一、视触觉夹爪（DM-Tac G）

### 1.1 产品概述

协作臂厂商A 视触觉夹爪（型号：DM-Tac G）是全新一代智能夹爪产品，实现了高精度夹持控制与多模态触觉感知深度融合。具备力位可编程、状态可视化、通讯灵活等先进机械夹爪优势，同时能实时采集接触形貌和高分辨率三维形变等丰富触觉信息，让机器人拥有如同人手般的灵敏"触感"和精准力控。

凭借感知与控制的协同优势，该产品能针对不同材质、形状与硬度的物体，灵活调整抓取策略，实现更稳定、更智能、更精巧的操作。

### 1.2 核心功能亮点

| 功能 | 说明 |
|------|------|
| **三维全分辨率触觉感知** | 切向与法向均具备超高分辨率触觉感知能力，可捕捉细微接触变化，全面提升感知精度 |
| **精准力位控制** | 支持夹持位置与夹持力值的可编程调节，结合力控与位控进行夹持判断，实现更智能、更稳固的抓取 |
| **多协议通讯兼容** | 内置 Modbus-RTU 与 IO 控制模式，可通过协议转换器扩展至 USB、EtherCAT、CAN、TCP/IP 等多种通讯方式 |
| **超高分辨率多模态感知** | 双指夹爪集成高精度视触觉传感器，以 120Hz 高频率实时采集数据（1秒采集超过900万组数据） |
| **全方位触觉信息获取** | 精确感知物体的三维形貌与接触面的三维形变，为软硬度识别、位姿识别、抓取位置优化及滑动检测等任务提供丰富可靠的数据支持 |

## 二、可穿戴遥操作数据采集设备（DM-EXton2）
