## 一、系统架构

### 1.1 云边端协同指控平台

```
┌─────────────────────────────────────────────────────────┐
│                    云平台（统一管理）                      │
│         路径规划 / 编队协同 / 任务规划 / 数据上云           │
└─────────────────────┬───────────────────────────────────┘
                      │ 5G / 工业WiFi
        ┌─────────────┼─────────────┐
        ▼             ▼             ▼
   ┌─────────┐   ┌─────────┐   ┌─────────┐
   │ 边缘节点 │   │ 边缘节点 │   │ 边缘节点 │
   │(方舱/终端)│   │(方舱/终端)│   │(方舱/终端)│
   └────┬────┘   └────┬────┘   └────┬────┘
        │             │             │
        └─────────────┼─────────────┘
                      ▼
         ┌────────────────────────────┐
         │ 无人平台（端）              │
         │ 四足机器人 / 无人车 / 无人机  │
         │ 人形机器人 / 巡检机器人      │
         └────────────────────────────┘
```

**关键能力:**
- 云平台统一管理 + 分布式控制（移动方舱）双模式
- 支持私有云 / 公有云 / 本地部署
- 集群协同：6+机器人同时巡检

### 1.2 核心算法模块

| 算法 | 功能 | 建筑场景应用 |
|------|------|-------------|
| 多源融合目标检测 | 可见光+红外+激光雷达融合识别 | 墙面裂缝/渗漏检测 |
| 云边端协同指控 | 边缘设备本地AI推理 | 现场实时决策 |
| 路径规划 | 动态障碍规避 + 最优路径 | 室内巡检路径 |
| 编队协同 | 多机器人任务分配 | 施工现场多机协同 |
| 人机交互 | 语音/手势/VR遥操作 | 远程专家指导 |
| SLAM地图 | 实时建图 + 指点寻路 | 未知环境探索 |

## 三、建筑场景专项应用

### 3.1 建筑施工场景映射

| 建筑AI任务 | 对应无人装备能力 | 实现方式 |
|-----------|-----------------|---------|
| **墙面钻孔检测** | 视觉识别+激光测距 | 云台相机+AI算法 |
| **施工进度监测** | 多机器人协同巡检 | 路径规划+3D测绘 |
| **危险区域预警** | 多元数据样本学习+险情识别 | 红外+雷达融合 |
| **材料搬运** | 编队协同+目标定位 | 无人车+机械臂 |
| **BIM对比验收** | 双目深度+SLAM建图 | 三维空间测绘 |
| **密闭空间检测** | 气体传感器+遥操作 | 防爆机器人 |
| **墙面裂缝/渗漏** | 多光谱图像AI识别 | 红外+可见光融合 |

### 3.2 多机器人巡检任务类型

```
任务类型          检测内容                    传感器
─────────────────────────────────────────────────────
施工进度状态监测    人员/材料/设备位置           可见光
三维空间测绘        空间尺寸 vs BIM模型          双目+激光雷达
危险源预警          裂缝/渗漏/异常热区           红外+视觉AI
阀门/搬运目标定位    物体位置+姿态识别            深度相机+视觉
```

## 四、系统集成方案

### 4.1 建筑机器人改装接口

无人装备指控平台可集成以下建筑机器人：
- 四足机器人（巡检/搬运）
- 无人车（室内外运输）
- 人形机器人（精细操作：钻孔/安装）

## 六、与建筑Physical AI系统的集成建议

### 6.1 接入方式

```
建筑Physical AI系统
    │
    ├── 世界模型（场景感知）
    │       │
    │       ▼
    ├── Harness层（安全约束：承重墙/管线）
    │       │
    │       ▼
    ├── Psi-R2 VLA（任务规划）
    │       │
    │       ▼
    └── 无人装备指控平台 ←── 接入点：/api/agent/process
            │

```

