## 🤖 机器人任务知识

### 任务1：搬运木地板

**感知需求：**
- 识别地板类型（实木/复合/SPC）
- 测量尺寸（长×宽×厚）
- 估计重量（密度×体积）

**抓取策略：**
| 地板类型 | 推荐方式 | 力度 | 理由 |
|----------|----------|------|------|
| 实木长板(>1.5m) | 双吸盘+支撑 | 中等 | 防翘曲断裂 |
| 实木短板 | 单吸盘 | 中等 | 标准抓取 |
| 强化复合 | 吸盘/夹具 | 轻 | 较轻 |
| SPC石塑 | 双吸盘 | 重 | 密度高、重 |

**搬运姿态：**
- 长板(>1.2m)：竖直搬运，减少弯矩
- 短板：水平搬运
- 多块堆叠：注意摩擦，不超过5块

### 任务2：安装木地板

**安装流程：**
```
1. 检查基层平整度 (<3mm/2m)
2. 铺设防潮膜
3. 确定铺设方向（顺光/顺长边）
4. 第一排：企口朝墙，留伸缩缝(8-12mm)
5. 逐排锁扣拼接
6. 最后一排：裁切、安装
7. 安装踢脚线
```

**机器人关键能力：**
- 锁扣对准精度：<0.5mm
- 敲击力度：50-100N（锁扣嵌入）
- 裁切精度：±0.5mm
- 伸缩缝预留：8-12mm

### 任务3：质量检测

**检测项目：**
| 项目 | 方法 | 合格标准 |
|------|------|----------|
| 平整度 | 2m靠尺 | <3mm |
| 缝隙 | 塞尺 | <0.3mm |
| 空鼓 | 敲击听音 | <5%面积 |
| 色差 | 视觉对比 | 同批次一致 |
| 损伤 | 视觉检测 | 无明显划痕 |

## 📐 物理计算示例

### 示例1：计算抓取力

```python
# 实木地板参数
length = 1.8  # m
width = 0.15  # m
thickness = 0.018  # m
density = 750  # kg/m³
mu_s = 0.6  # 静摩擦系数(橡胶吸盘)

# 计算质量
volume = length * width * thickness  # 0.00486 m³
mass = density * volume  # 3.645 kg

# 计算最小抓取力
g = 9.8  # m/s²
F_grip_min = mass * g / (2 * mu_s)  # 29.8 N

# 安全系数 2.0
F_grip_safe = F_grip_min * 2.0  # 59.6 N
```

### 示例2：计算堆叠稳定性

```python
# 堆叠参数
n_planks = 5  # 块数
mass_per_plank = 3.6  # kg
mu_plank_plank = 0.5  # 地板间摩擦系数

# 总重量
total_mass = n_planks * mass_per_plank  # 18 kg

# 最底层摩擦力
F_normal = total_mass * 9.8  # 176.4 N
F_friction = mu_plank_plank * F_normal  # 88.2 N

# 水平加速度限制
a_max = F_friction / total_mass  # 4.9 m/s²
# 机器人搬运加速度应 < 4.9 m/s²
```
