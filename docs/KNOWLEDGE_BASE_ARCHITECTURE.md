## 核心架构：五层金字塔

```
┌─────────────────────────────────────────────────────────────┐
│                     L5: 应用层                               │
│    机器人导航 | 家务服务 | 安防监控 | 设施维护                │
├─────────────────────────────────────────────────────────────┤
│                     L4: 场景层                               │
│    家庭住宅 | 商业办公 | 园区景观 | 公共空间                  │
├─────────────────────────────────────────────────────────────┤
│                     L3: 空间层                               │
│    客厅 | 卧室 | 厨房 | 卫生间 | 阳台 | 走廊 | 庭院          │
├─────────────────────────────────────────────────────────────┤
│                     L2: 对象层                               │
│    建筑结构 | 家具 | 家电 | 软装 | 机电设施                  │
```

## 二、类型维度（纵向分类）

```yaml
对象分类体系:

  1. 建筑结构 (Structure):
    墙体:
      - 承重墙: 质量=2400kg/m³, 刚度=30000MPa, 不可拆改
      - 隔墙: 质量=800kg/m³, 刚度=8000MPa, 可调整
      - 剪力墙: 抗震结构，严禁拆改
      
    地板:
      - 瓷砖: 摩擦系数=0.5, 硬度=硬
      - 木地板: 摩擦系数=0.4, 硬度=中
      - 大理石: 摩擦系数=0.6, 硬度=极硬
      
    天花:
      - 吊顶: 高度、材质、承重
      - 梁位: 结构梁，高度限制
      
    门窗:
      - 门: 平开门、推拉门、折叠门
      - 窗: 落地窗、飘窗、天窗

  2. 家具类 (Furniture):
    客厅:
      - 沙发: 三人位、单人位、贵妃椅
      - 茶几: 实木、玻璃、金属
      - 电视柜: 悬挂式、落地式
      - 书架: 开放式、封闭式
      
    卧室:
      - 床: 双人床、单人床、榻榻米
      - 衣柜: 推拉门、平开门
      - 床头柜: 配套尺寸
      
    餐厅:
      - 餐桌: 长桌、圆桌
      - 餐椅: 匹配数量
      
    书房:
      - 书桌: 尺寸、材质
      - 书架: 容量计算

  3. 家电类 (Appliance):
    厨房:
      - 冰箱: 尺寸、功率、散热空间
      - 洗衣机: 尺寸、水电路
      - 微波炉: 安装高度
      - 烤箱: 嵌入式尺寸
      - 油烟机: 排烟管道
      
    客厅:
      - 电视: 尺寸、重量、安装高度
      - 空调: 挂机、柜机、中央空调
      
    卧室:
      - 空调: 匹配面积
      - 加湿器: 容量
      
    卫生间:
      - 热水器: 电热、燃气
      - 浴霸: 安装位置

  4. 软装类 (Soft Furnishing):
    织物:
      - 窗帘: 材质、遮光率
      - 地毯: 尺寸、材质
      - 抱枕: 数量、尺寸
      
    装饰:
      - 挂画: 尺寸、高度
      - 绿植: 光照需求
      - 摆件: 易碎程度
      
    灯具:
      - 吊灯: 安装高度
      - 台灯: 摆放位置
      - 壁灯: 安装高度

  5. 机电设施 (Utility):
    电气:
      - 开关: 高度130cm（国标）
      - 插座: 高度30cm/110cm
      - 配电箱: 回路分配
      - 弱电箱: 网络布线
      
    水路:
      - 水龙头: 冷热水分路
      - 角阀: 控制点位
      - 排水: 坡度要求
      
    暖通:
      - 地暖: 分水器位置
      - 新风: 出风口
      - 中央空调: 风管走向
      
    安防:
      - 监控: 覆盖范围
      - 门禁: 位置
      - 报警器: 烟雾、燃气
```

## 六、知识图谱（语义网络）

```yaml
知识图谱结构:

  空间关系图谱:
    包含关系:
      - 房屋 → 楼层 → 房间 → 区域
      
    相邻关系:
      - 客厅 ↔ 餐厅: 开放连接
      - 厨房 ↔ 阳台: 门连接
      - 卧室 ↔ 卫生间: 门连接
      
    上下关系:
      - 一楼 ↔ 二楼: 楼梯连接
      
  功能关系图谱:
    供电关系:
      - 配电箱 → 回路 → 开关 → 灯具/插座
      - 回路标注: L1主灯、L2筒灯、L3氛围灯
      
    供水关系:
      - 入户总阀 → 支管 → 角阀 → 终端
      - 冷热分离: 热水器 → 热水管路
      
    供气关系:
      - 燃气表 → 支管 → 灶具/热水器
      - 阀门位置: 燃气表旁、灶具下
      
  物理关系图谱:
    承重关系:
      - 楼板 → 承重墙 → 基础
      - 不可拆改墙体标识
      
    传热关系:
      - 外墙 → 保温层 → 内墙
      - 热桥位置标注
      
    声学关系:
      - 隔音墙位置
      - 传声路径分析
```

## 九、查询接口设计

### 9.1 项目查询

```python
# 查询项目
GET /api/projects?category=villa&location=广州

# 获取项目详情
GET /api/projects/{project_id}

# 获取项目图片
GET /api/projects/{project_id}/images?type=render

# 获取项目空间
GET /api/projects/{project_id}/spaces
```

### 9.2 对象查询

```python
# 查询对象
GET /api/objects?category=furniture&type=sofa

# 获取对象详情
GET /api/objects/{object_id}

# 获取物理属性
GET /api/objects/{object_id}/physics

# 获取关联对象
GET /api/objects/{object_id}/relations
```

### 9.3 场景查询

```python
# 查询场景
GET /api/scenes?type=cleaning

# 获取场景任务
GET /api/scenes/{scene_id}/tasks

# 获取训练数据
GET /api/scenes/{scene_id}/training_data
```
