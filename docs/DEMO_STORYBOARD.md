## 第1幕：自然语言指令（3-15秒）

| 属性 | 内容 |
|------|------|
| 时长 | 12秒 |
| 镜头 | 屏幕录制UI，左侧3D场景，右侧任务面板 |
| 画面 | 鼠标点击输入框，输入文字："在东墙钻孔挂画" |
| UI动画 | 输入框发光，文字逐渐消失→任务卡片弹出 |
| 文字标签 | 系统解析过程可视化：位置→墙体→工具→顺序 |
| 旁白 | *"自然语言，即刻转化为建筑级操作方案。"* |

**解析动画（5秒）：**
- 「东墙」→ 🔴 承重墙检测中...
- 「钻孔」→ 📐 工具选择：电钻 + 水平仪
- 「挂画」→ 📏 深度计算：25mm，避开管线

```

├─────────────────────────────────────────┤
│  🟢 材质分析 — 非承重墙                  │
│     材质：加气混凝土砌块                  │

├─────────────────────────────────────────┤
```

**3D场景联动：**
- 红色高亮「东墙」
