## 前端接入

- **场景 ID**: `renderings`
- **入口**: 控制面板 → 室内/园林/建筑 分类 → 选择「效果图库」
- **组件**: `RenderingGallery.tsx` - 缩略图画廊 + 预览
- **效果**: 右侧显示缩略图网格，支持分类/场景过滤，支持点击大图预览
