# Demo 录制指南 — Physical AI 世界模型 (2026-04-30)

## 录制环境准备

### 启动方式（二选一）

**方式A：直接运行 exe（推荐）**
```
electron\dist-electron\win-unpacked\Physical AI Spatial World Model.exe
```

**方式B：前端开发模式（更流畅）**
```powershell
cd projects\building-physical-ai\web-app
npm run dev
# 浏览器打开 http://localhost:3000
# 后端服务需单独启动
```

### 服务启动顺序
1. 双击 exe → 等待 5 秒 → 5 个服务自动拉起
2. 打开浏览器 → http://localhost:3000
3. 验证 5 个服务全部在线（TopBar 绿色）

## 录制技巧

### 画面设置
- **分辨率**：1920×1080（全屏录制）
- **帧率**：30fps 足够，60fps 更流畅
- **光标**：录制时隐藏光标，或用聚光灯效果

### 语速
- 总时长 3 分钟，配音约 500 字
- 重点部分（第4、5幕）语速放慢，强调技术壁垒

### 背景音乐
- 建议：轻科技风 BGM（音量-20dB）
- 推荐： Epidemic Sound / Artlist 免费曲目

### 推荐录制工具
| 工具 | 优点 |
|------|------|
| OBS Studio（免费） | 功能全，支持窗口/全屏/区域录制 |
| Bandicam | 压缩率高，适合长时间录制 |
| PowerPoint录制 | 简单快捷，适合PPT风格 |

### 快捷操作提醒
- **提前演练 2 遍**：确保服务启动顺序、场景切换流畅
- **准备 fallback**：如果某服务启动失败，知道如何快速重启
- **关闭无关应用**：避免通知弹出干扰画面

## 故障排查

### 服务未全部启动
```powershell
# 检查端口占用
netstat -ano | findstr "5000 5001 5002 5004 8888"
# 重启 exe
```

### 前端白屏
- 刷新浏览器（Ctrl+Shift+R 强制刷新）
- 检查 TopBar 服务状态

### Agent 无响应
- 确认 5002 端口在线
- 查看 exe 日志窗口输出
