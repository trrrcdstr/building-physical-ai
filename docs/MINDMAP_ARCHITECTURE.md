## 数据层 — 效果图接入

| 字段 | 说明 | 来源 |
|------|------|------|
| `id` | 唯一标识 `室内_家庭_0001` | 自动生成 |
| `name` | 效果图原始名称 | 文件名 |
| `category` | 室内/室外/建筑 | 目录分类 |
| `scene` | 家庭空间/商业空间等 | 子目录+规则 |
| `scene_type` | residence/mall/office等 | 映射 |
| `tags` | 风格+空间类型标签 | 文件名关键词 |
| `path` | 原始文件路径 | 扫描 |
| `position` | 虚拟3D坐标 | 哈希分布 |
