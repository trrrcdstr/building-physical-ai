## 🧠 常识知识体系

### 物理常识
- **力学**：重力、摩擦、碰撞、弹性 → 抓取、搬运、放置
- **材质**：硬度、脆性、柔性、吸水性 → 分类、处理力度
- **温度**：耐热、耐寒、燃点 → 厨房、消防
- **化学**：酸碱、腐蚀、反应 → 清洁剂、污水处理

### 空间常识
- **布局**：房间功能、动线设计 → 路径规划
- **尺寸**：门宽、通道、货架高度 → 通行判断
- **层次**：地面、墙面、吊顶、台面 → 操作空间
- **边界**：台阶、门槛、坡度 → 安全行走

### 社会常识
- **礼仪**：敲门、让路、称呼 → 服务场景
- **安全**：禁区、危险品、应急出口 → 安防巡检
- **隐私**：卧室、卫生间、办公区 → 入户服务
- **时间**：高峰期、夜间模式、节假日 → 任务调度

## 🤖 具身智能能力要求

### 感知能力
- **视觉**：物体识别、场景理解、缺陷检测
- **触觉**：力觉感知、材质识别、滑移检测
- **听觉**：语音指令、异常声响、环境感知
- **位置**：室内定位、地图构建、导航

### 运动能力
- **移动**：行走、避障、乘梯
- **操作**：抓取、放置、旋转、推拉
- **负重**：搬运、举升、推车
- **精细**：拧螺丝、按开关、写字
